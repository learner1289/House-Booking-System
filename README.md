<h1 align="center">House Booking System</h1>

### Overview
House Booking System is a console-based application developed in C for managing house rentals and bookings across the country. The project allows administrators to manage houses and bookings, while customers can search and reserve available houses easily. 
This was an academic capstone project completed during the 2nd year of undergraduate studies.

### Tech Stack
- Language: C
- Application Type: Console-Based Application
- Concepts Used: File Handling, Structures, Functions, Conditional Statements, Loops

### Project Structure

```bash
House-Booking-System/
├── House_Booking_System.c

```
### Features
**Admin Functionalities**
- Add New House
- View All Houses
- View All Bookings
- Delete House
- Modify House Details

**Customer Functionalities**
- View Available Houses
- Search Houses
- Book a House
- View My Bookings
- Cancel Booking
